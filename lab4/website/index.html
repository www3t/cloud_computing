What We Learned in Lab 4

Key Amazon S3 concepts covered in this laboratory work

🪣

S3 Buckets

Created public and private buckets with proper ACL settings. Public bucket hosts avatars and static content, private bucket stores logs with lifecycle policies.

🔐

Access Control

Configured Block Public Access settings and ACLs. Public objects are accessible via URL, private objects return 403 Forbidden.

⚙️

AWS CLI

Used aws s3 cp, mv and sync commands to upload and manage objects. Applied --acl public-read flag for public access.

🔄

Versioning

Enabled bucket versioning to track all changes to objects. Every upload creates a new version, allowing rollback to previous states.

📅

Lifecycle Rules

Configured automated lifecycle: Standard-IA after 30 days, Glacier Deep Archive after 365 days, deletion after 5 years.

🌐

Static Hosting

Deployed this website using S3 Static Website Hosting — no EC2 needed. Perfect for HTML/CSS/JS sites and React apps.
